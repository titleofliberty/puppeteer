Puppeteer

Puppeteer is an application to assist players in desktop roll playing. It is meant to be as minimally distracting as possible to allow the player to focus on the game. It is designed to be used with a keyboard as much as possible, allowing for quick and efficient gameplay, without having to switch between the keyboard and mouse. In an effort to be game system agnostic, Puppeteer refers to the player's character as a "puppet".

At present, Puppeteer only supports the Dungeons and Dragons™ 5th Edition ruleset, but support for other rulesets may be added in the future.
